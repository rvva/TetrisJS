## Run it with docker engine!
1. Clone repository.
```
cd /srv
git clone https://github.com/rvva/TetrisJS/
```
2. Go to the project directoy and run docker-compose.
```
cd /srv/TetrisJS/
docker-compose up -d
```
3. That's it. Game will be available at localhost, port 80.
